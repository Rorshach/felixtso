LinkVan (Fall 2015 - Present)

  • Built on Ruby on Rails, html, css, javascript, jQuery and bootstrap.
  • Web app to help members of the Vancouber Downtown Eastside find resources such as food, shelters, and showers
  • The project started with a UBC grant and a partnership with the UBC Learning Exchange.
  • I'm currently the lead developer of the project and my main work focuses on maintaining the app through bug fixes and adding new features to the app requested by the community.
  • Some of the features include user sign up, new facility form, new resources on the home page, and admin user to verify users and facilities.
  • We've also partnered with SFU to research how adults with low literacy levels use technology.
  • Here's some of the media coverage our team has received: Globe and Mail, Metro News, Current Grant
  • LinkVan
